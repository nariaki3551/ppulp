Welcome to ppulp's documentation!
=================================

P(retty)PuLP is developed as an extension of the linear programming modeling tool `PuLP <https://coin-or.github.io/pulp/>`_ .
PPuLP has following functions.

- Variable product
- If-then constraints
- Absolute values
- Piecewise linear approximation of nonlinear functions
- Logical operations (And, Or, Xor)
- Reduction (Sum, Prod)

PPuLP helps user to create modeling more intuitive and simplified.
